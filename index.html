Probability

üé≤ Basics
Update Rule
$\theta:=\theta-\alpha\nabla J(\theta)$
Note: \(\alpha\) is the learning rate.
üöÄ Advanced Optimizers
Momentum
$v:=\beta v+(1-\beta)\nabla J(\theta)\\\n\theta:=\theta-\alpha v$
Adam (simplified)
$m:=\beta_1 m+(1-\beta_1)\nabla J\\\nv:=\beta_2 v+(1-\beta_2)(\nabla J)^2\\\n\theta:=\theta-\alpha\frac{\hat m}{\sqrt{\hat v}+\epsilon}$
Pro tip: Adam is widely used for deep learning.

Clustering

üéØ K-Means
Objective
$\min \sum_{j=1}^k\sum_{x\in C_j}\|x-\mu_j\|^2$
Centroid
$\mu_j=\frac{1}{|C_j|}\sum_{x\in C_j}x$
üìè Distance Metrics
MetricFormula
Euclidean$d(x,y)=\sqrt{\sum_i (x_i-y_i)^2}$
Manhattan$d(x,y)=\sum_i|x_i-y_i|$
Cosine$\cos(\theta)=\frac{x\cdot y}{\|x\|\|y\|}$
üìä Silhouette Score
Silhouette Coefficient
$s(i)=\frac{b(i)-a(i)}{\max\{a(i),b(i)\}}$
Where: \(a(i)\) = mean distance to same cluster, \(b(i)\) = mean distance to nearest cluster

Deep Learning

üß† CNN / RNN / LSTM
Conv Output Size
$O=\left\lfloor\frac{W-K+2P}{S}\right\rfloor+1$
Parameters: W = input size, K = kernel size, P = padding, S = stride
LSTM Forget Gate
$f_t=\sigma(W_f[h_{t-1},x_t]+b_f)$
LSTM Input Gate
$i_t=\sigma(W_i[h_{t-1},x_t]+b_i)$
LSTM Cell State
$C_t=f_t\odot C_{t-1}+i_t\odot\tilde{C}_t$
üíß Batch Norm / Dropout
BatchNorm (normalize)
$\hat x=\frac{x-\mu_B}{\sqrt{\sigma_B^2+\epsilon}}$
Scale & Shift
$y=\gamma\hat x+\beta$
Dropout: During training, randomly set activations to 0 with probability \(p\). During testing, use all neurons.

Regression

üìâ Linear Regression
Model
$$y=\beta_0+\beta_1 x+\varepsilon$$
Matrix Form
$$\mathbf{y}=\mathbf{X}\beta+\varepsilon$$
Normal Equation
$$\beta=(X^TX)^{-1}X^T y$$
MSE
$$MSE=\frac{1}{n}\sum_{i=1}^n (y_i-\hat{y}_i)^2$$
R¬≤
$$R^2=1-\frac{\sum(y_i-\hat{y}_i)^2}{\sum(y_i-\bar{y})^2}$$

Sigmoid
$$\sigma(z)=\frac{1}{1+e^{-z}}$$
Log Loss
$$L=-\frac{1}{n}\sum_{i=1}^n[y_i\log\hat{y}_i+(1-y_i)\log(1-\hat{y}_i)]$$
üõ°Ô∏è Regularization

Elastic Net$$J(\beta)=\sum(y_i-\hat{y}_i)^2+\lambda_1\sum|\beta_j|+\lambda_2\sum\beta_j^2$$
